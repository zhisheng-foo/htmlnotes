<input> tag























Note the use of the autofocus attribute for input type text. The autofocus attrbute is used to automatically focus a particular form element when the page loads. This means that as soon as the page is loaded into the browser, the specified element (such as an input field, button, or textarea) becomes the active focus and is ready for user input without the user having to click on it.
